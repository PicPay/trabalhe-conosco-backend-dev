
### INSTALAÇÃO

Após iniciar o Docker no sistema operacional, digite todos os comandos abaixo no diretório raiz do projeto.

- composer install
- docker-compose up
- docker-compose run app php artisan migrate
- docker-compose run app php artisan import:db
