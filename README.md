
# INSTALAÇÃO

Após iniciar o Docker no sistema operacional, digite todos os comandos abaixo no diretório raiz do projeto.

1° - composer install
2° - docker-compose up
3° - docker-compose run app php artisan migrate
4° - docker-compose run app php artisan import:db
